## Top 3 ‚Äî fix these first
1. **[HIGH]** `decrypt()` leaks an uncaught `RecursionError` on a deeply-nested JSON header ‚Äî a malformed/hostile file crashes with a raw traceback instead of the documented clean `None`. Add `RecursionError` (and/or pre-limit header size/nesting) to the malformed-envelope handler + regression test.
2. **[MED]** CLI ignores the `lock`/`unlock` return value ‚Üí a failed `unlock` (wrong password) exits `0`, so scripts/pipelines can't detect failure. Capture the result and `sys.exit(1)` on failure.
3. **[MED]** `_atomic_write` silently drops the original file's mode/owner/group (mkstemp `0600` + `os.replace` ‚Üí every round-trip tightens perms to owner-only). `copystat` from the original onto the temp before replace, or document the intentional tightening.
